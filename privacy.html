Privacy Policy

K Creator Toolbox의 개인정보 보호 방침

1. 개요

본 사이트는 프라이버시 보장을 최우선으로 하며,
사용자의 파일이나 데이터는 서버로 전송되지 않고 브라우저 내에서만 처리됩니다.

2. 수집하는 정보

  • 파일: 업로드된 파일은 로컬에서만 처리되며 서버에 저장되지 않습니다.
  • 계정: 회원가입이나 로그인 기능을 제공하지 않으므로 계정 정보를 수집하지 않습니다.
  • 댓글: 댓글 기능(utterances)은 GitHub Issues 기반으로 동작하며, GitHub의 정책이 적용됩니다.

3. 제3자 서비스

본 사이트는 아래와 같은 외부 서비스를 사용합니다.

  • GitHub Pages – 사이트 호스팅
  • Utterances – 댓글 기능 (GitHub 계정 필요)
  • Google Fonts – 웹 폰트 제공

4. 쿠키 및 로컬 저장소

서비스 이용을 위해 필수적인 환경 설정(예: 다크 모드, 언어 선택)은
사용자의 브라우저 로컬 저장소에만 저장됩니다.

5. 이용자의 권리

사용자는 GitHub Issues에 남긴 댓글을 직접 수정·삭제할 수 있으며,
기타 개인정보 관련 문의는 아래 연락처로 요청하실 수 있습니다.
