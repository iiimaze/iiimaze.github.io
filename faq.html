FAQ

자주 묻는 질문 모음

Q1 서버로 업로드되나요?

아니요. 모든 처리는 브라우저 로컬에서만 이루어집니다.
파일은 서버로 전송·저장되지 않습니다.

Q2 용량이 너무 커요

출력 용량을 줄이려면 아래를 시도해 보세요.

  • 해상도를 낮추거나 목표 사이즈를 지정
  • 프레임레이트(영상일 경우) 낮추기
  • 품질(quality) 값을 단계적으로 낮추기

Q3 개인정보가 저장되나요?

회원 시스템을 사용하지 않으므로 계정 정보를 수집하지 않습니다.
댓글은 GitHub Issues(utterances) 정책을 따릅니다.
